# Excel Sales Dashboard

This project is an interactive sales dashboard created using Microsoft Excel.

## Dataset Columns
- OrderID
- Date
- Region
- Salesperson
- Product
- Units
- Revenue
- Month
- Year
- Sales Category

## Key Features
- Dynamic KPI cards (Total Revenue, Total Units, Total Orders, Average Revenue)
- Interactive slicers for Region, Year, and Sales Category
- Pivot Tables used as backend for calculations
- Clean dashboard layout for business reporting

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Excel Charts
